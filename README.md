# goalie-bot
 A game where one player shoots, and the other blocks

## What this is

This started off as an idea for an automated AI goalie that will save all shots made on a miniature soccer goal, but then as the hackathon hour requirement got canceled, i decided to change the idea. It's now a 2-player game where one person shoots, and the other tries to block the penalty kick by using 2 buttons to dive left or right. 

## How it's made

I made this using a leftover Raspberry Pi Pico W i had lying around, along with a NEMA 17 motor. The motor can't just be controlled by the raspberry pi, though, so i bought an A4988 motor controller to control it. I also needed something to power it, so i used this 12v power supply. I needed something to mount this all on, so i bought some wires and a couple of breadboards. The Joystick and LED Matrix Displays were just side projects i was hoping to get functioning, but the Button Switches are part of the final design
